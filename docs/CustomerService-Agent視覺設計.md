## 六、響應式設計

### 6.1 斷點

| 斷點 | 客服儀表板布局 |
|------|--------------|
| ≥ 1440px | 全三欄（流程 280px + 列表 彈性 + 明細 320px）|
| 1024 ~ 1439px | 兩欄（流程+列表合併），明細抽屜 |
| 768 ~ 1023px | 單欄，明細全屏覆蓋 |
| < 768px | 行動優先：卡片式堆疊，流程圖縮為時間軸 |

### 6.2 行動版時間軸

手機版將流程圖改為垂直時間軸：
```
📥 詢單接收    09:30 ✅
     ↓
📋 PO 建立     09:45 ✅
     ↓
🔍 ATP/CTP    10:00 🔄
     ↓
...
```

## 七、無障礙設計

| 項目 | 規範 |
|------|------|
| 色彩對比 | 文字與背景對比 ≥ 4.5:1（WCAG AA）|
| 狀態標識 | 不只靠顏色，搭配圖標 + 文字標籤 |
| 鍵盤導航 | 所有互動元素可 Tab 聚焦 |
| 焦點指示 | `outline: 2px solid #3B82F6` |
| 動效偏好 | 尊重 `prefers-reduced-motion`，減速或關閉 |
