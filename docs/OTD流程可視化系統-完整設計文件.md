## 一、系統概述

### 1.1 設計願景

建立一個可視化的模擬工廠接單到出貨（OTD）完整流程系統，以 OpenClaw Agent 作為事件驅動的智能處理核心，透過 Webhook 串接每個流程節點，實現從詢單到出貨的全鏈條自動化與可視化。

### 3.2 Webhook 端點一覽

| 步驟 | Webhook 端點 | 觸發事件 | 負責 Agent |
|------|-------------|---------|-----------|
| ①詢單 | `POST /webhook/inquiry/received` | 新詢單建立 | CS Agent |
| ②PO | `POST /webhook/po/received` | PO 接收確認 | CS Agent |
| ③ATP/CTP | `POST /webhook/atp-ctp/request` | 交期試算請求 | ATP/CTP Agent |
| ③ATP/CTP | `POST /webhook/atp-ctp/confirmed` | 試算完成 | ATP/CTP Agent |
| ④SO | `POST /webhook/po-so/convert` | PO 轉 SO | PO-to-SO Agent |
| ④SO | `POST /webhook/so/created` | SO 建立 | PO-to-SO Agent |
| ⑤追蹤 | `POST /webhook/logistics/track` | 進度查詢 | Logistics Agent |
| ⑥ASN | `POST /webhook/asn/created` | 出貨預告 | CS Agent |
| ⑦出貨 | `POST /webhook/pick-pack/ship` | 出貨作業 | ERP 模擬層 |
| ⑧Invoice | `POST /webhook/invoice/sent` | 發票發出 | CS Agent |
| ⑨CV | `POST /webhook/customer-voice/received` | 客戶回饋 | After Service Agent |
